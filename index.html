To Use

Add CSS and JavaScript to <HEAD> (Download)

<link rel="stylesheet" href="marquee-light.css">
<script src="marquee-light.js"></script>

Create the marquee in HTML <BODY>

<div class="marquee-instance">
  <div class="marquee">
    <div class="marquee-track">
      <div class="marquee-content" data-content-source>
        <!-- Your content here -->
        <div>Item 1</div>
        <div>Item 2</div>
        <div>Item 3</div>
      </div>
    </div>
  </div>
  <div class="marquee-controls">  
    <button class="toggle-marquee-btn" title="Stop marquee">
      Stop marquee
    </button>
  </div>
</div>

Atrributes

Configure each maruqee individually with data attributes. 

<div class="marquee-instance" data-speed="0.2" data-gap="1rem" data-fade="false">


Attribute Value Description
data-speed 0.1 - 1 (Recommended) Set the speed of the marquee
data-gap 0rem - 2rem (Recommended) Sets the distance (gap) between each item
data-fade true / false Add a graident fade to the start and end of the marquee track
